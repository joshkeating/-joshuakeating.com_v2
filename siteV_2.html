Site V2

November 23th, 2016

The primary motivation behind redoing this site was to make it scale for mobile correctly. With the help of bootstrap and media queries I was able to significantly increase the overall responsiveness. The site uses good 'ole html and css throughout and relies on some of the fancier features of css3 to pull of the slick transitions on the main page.

This was my first experience working with a web framework and while bootstrap is only front end, I plan on picking up a back end framework that I can use to build more dynamic web pages. Check out the source code here.

Implementing bootstrap

The easiest way to setup bootstrap is to include the CDN in the page's <head>. It looks something like this 

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/css/bootstrap.min.css">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/1.12.4/jquery.min.js"></script>
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.7/js/bootstrap.min.js"></script>
This will load the framework in from cache when the page is visited and is generally faster than self-hosting bootstrap.
